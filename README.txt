This is the online appendix for the submitted paper for the SSCI conference 2017.

Inside each folder is another README that will explain the content and what information is being presented/analysed

The folder ReEval_PerMorph contains the box and whisker plots that show the re-evaluation results when each controller was tested using every other sensory configuration

The folder T_Test_Results contains the tables that indicate in which comparisons there were a statistically significant difference
